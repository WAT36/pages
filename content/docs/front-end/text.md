# テキスト

HTML文書にテキストを記載、装飾、補足するための要素についてをここでは述べる。

## p要素

p要素は、一つの段落を表す要素である。

```
<p>段落１</p>
<p>段落２</p>
```

## a要素

a要素は別URLへのリンクを示す時に使う要素であり、通常は**href**属性を指定して利用する。

href属性で遷移先のURLを指定する。

```
<p><a href="/">ホーム</a></p>
```

## em要素

em要素は強調(emphasize)する部分を示すための要素である。入れ子にすることもでき、それにより強調の度合いを強くすることもできる。

```
<p>ここが<em><em>超</em>タイヘン</em>な箇所だ</p>
```
