# NaN

**NaN**とは数値では無い値(Not a Number)のことを表す値で、プログラムで数値計算をする際に、正常な演算結果が得られなかったことを示す数値表現として用いられる。

例えば、0を0で除算した結果はNaNになる。（言語によっては、このような演算をした場合エラー出力をさせる場合もある）
