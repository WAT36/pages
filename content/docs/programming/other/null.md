# Null

プログラミングでよく見るNullについて。

Nullとは何なのか？

Nullとは何もない、何も入ってない、という状態を示す語である。

変数に何も値が入ってないとき、変数の値を参照しようとするとNullが返ってくる。

よく0と勘違いされることが多いが、変数の値を参照して0が返ってきたときは、0という値が変数に入っているという意味であり何も値が入っていないという意味ではない。
