# オブジェクト(Javascript)

Javascriptのオブジェクトについて。

Javascriptでは、オブジェクトと呼ばれる、C言語などの構造体のようなデータ構造が存在する。

ここではそれについて記載する。

## オブジェクトの定義

{{< tabs "for" >}}
{{< tab "Javascript" >}}

Javascriptでのオブジェクトは、以下のような方法で記載する。

記法

```
var 変数名 = {
    (key名1) : (value名1),
    (key名2) : (value名2),
    ・・・
}
```

このように、キーと値のセットを複数連ねたような構造になる。

ちなみに、オブジェクトではこのキーと値のセットのことを**プロパティ**と呼んでいる。
