# 辞書、Map

辞書、Mapについての記事。  

多くの言語でこれに似たようなデータ構造がある（と予想している）が、言語により名前が違うので注意。    
JavaではMap、Pythonでは辞書と呼ばれている。  
C言語では構造体、JavaScriptではオブジェクト(プロパティ)と言った名前のデータ構造で呼ばれている（これらを似た構造としてひとまとめにする事に否定的な意見もあるかもしれないが、ここでは一緒の候としてまとめておく）
