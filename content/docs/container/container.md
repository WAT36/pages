# コンテナとは？

最近よく聞くコンテナ技術とは何なのか？

コンテナとは、OS上に論理的な区画(コンテナ)を作り、そこにアプリケーションやアプリケーションを動作させるのに必要なライブラリなどをまとめ、そのコンテナ自体があたかも個別のサーバーのように使うことができるようにしたものである。

コンテナ技術を利用すると、OS、IPアドレスなどのリソースを、ここのアプリケーションがあたかも占有しているかのように見せることができる。
