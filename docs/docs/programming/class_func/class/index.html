    クラスの定義 #

    クラスを定義する方法についてを示す。

    Javaでのクラス定義方法は以下の通り。

    [public] class クラス名 [extends 継承元クラス名] [implements 実装先インタフェース名]{
    //文
}

    あるクラスを継承して作りたい時はクラス名の後にextendsを、インタフェースを実装する時はその後にimplementsを書き加える。

    Pythonでのクラスを定義するときの記法は以下の通り。

    class クラス名:
    //文
