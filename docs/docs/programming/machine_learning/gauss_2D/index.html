    ガウス関数(2次元)

    ガウス関数(2次元) #

    ガウス関数

    a exp ( - (x - μ ) 2 σ 2 )

    において、入力xを2次元ベクトルとした時の場合を考える。(以下)

    x = x 0 x 1

    この時、μも入力x0、x1の中心(平均)である要素μ0、μ1を持つベクトルを作成する。

    μ = μ 0 μ 1

    また、分散(標準偏差)σに関しては、ベクトルの要素間の分散である共分散行列を用いる。
    今回の2変数の場合だと以下のようになる。

    Σ = σ 00 σ 01 σ 10 σ 11

    σijはxの要素xi、xjの分散であり、σij = σji である。

    これらを利用し、2次元でのガウス関数は以下のようになる。

    y = a exp { - 1 2 ( x - μ ) T Σ -1 ( x - μ ) }
