    辞書、Map #

    辞書、Mapについての記事。

    多くの言語でこれに似たようなデータ構造がある（と予想している）が、言語により名前が違うので注意。
    JavaではMap、Pythonでは辞書と呼ばれている。
    Javascriptでは辞書,Mapといった名前のデータ構造は無いが、これに似たものとして連想配列と呼ばれるデータ構造が存在する。ここではこの連想配列についてを記載する。
