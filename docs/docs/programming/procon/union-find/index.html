    Union-Find木 #

    Union-Find木について。

    Union-Find木は要素のグループ分けを管理するデータ構造である。 1つ以上の要素を含むグループが何個かあった時、次の事が行える。

    • 2つのグループをマージする
    • 2つの要素が同じグループに含まれているか判定する

    注意としては、グループを分割することはできないという事。
