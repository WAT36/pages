    2変数関数を定義する

    2変数関数の定義は関数定義の時に引数を２つ設定すれば良い。
    試しに、f(x,y) = √(x2 + y2)を定義する例を示す。

    >>> import math
>>> import numpy as np
>>> 
>>> def f(x,y):
...     return math.sqrt(x**2 + y**2)
... 
>>> 
>>> x = np.linspace(-3,3,61)
>>> y = np.linspace(-3,3,61)
>>> 
>>> z = np.zeros((len(x),len(y)))
>>> 
>>> for xi in range(len(x)):
...     for yi in range(len(y)):
...             z[yi,xi] = f(x[xi],y[yi])
... 
>>> 
>>> print(z)
