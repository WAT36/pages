    グラフ・データ処理 #

    データサイエンスでは必須？ グラフ描画やcsv等のデータ前処理についてをまとめる。 ここも都合上、現在はPythonのみ記載。

    グラフ描画ではmatplotlibのpyplotというライブラリを利用する。

    • グラフ
      • 2次元グラフ
        • (折れ線)グラフを描画する
        • 一変数関数f(x)をグラフに描画する
        • グラフに色をつける
        • グラフの凡例を表示する
        • x,y軸の表示範囲を指定する
        • グラフのタイトルを設定する
        • 軸のラベルを設定する
        • グラフにグリッド線を表示させる
        • グラフを複数表示させる
      • 3次元グラフ
        • 2変数関数を定義する
        • 2変数関数のグラフ(ヒートマップ)を作成する
        • 2変数関数のグラフ(３D面グラフ)を作成する
