    逆行列

    逆行列を算出する方法についてを述べる。

    逆行列を算出したいときは、numpy.linalgモジュールにある関数invを使用する。

    numpy.linalg.inv(A)

    上記の式により、行列Aの逆行列が生成される。
    実行例を以下に示す。

    >>> import numpy as np
>>> 
>>> #行列Aの定義
>>> a=np.array([[1,2],[3,4]])
>>> print(a)
[[1 2]
 [3 4]]
>>> 
>>> #行列Aの逆行列
>>> print(np.linalg.inv(a))
[[-2.   1. ]
 [ 1.5 -0.5]]
>>> 
>>> #行列B（逆行列は存在しない）
>>> b=np.array([[1,1],[1,1]])
>>> print(b)
[[1 1]
 [1 1]]
>>> 
>>> #Bの逆行列 -> 逆行列は存在しないのでエラー
>>> print(np.linalg.inv(b))
