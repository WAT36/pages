    イベント

    Javascriptのイベントについてを記す。

    イベントとは、ユーザーの画面上での何らかの操作を、プログラム上で検知し、利用できるようにする機能である。

    画面上での操作とは、例えばあるボタンを押した、あるテキストをフォーカスした、などが挙げられる。
