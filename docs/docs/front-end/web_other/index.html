    XHTML

    XHTMLとは、HTMLをXMLの文法で定義し直して書かれた文書である。

    HTMLとの相違点は以下の通り。

    • 文書の先頭にXML宣言文が必要
    • 文字の大文字・小文字が区別される
    • 要素は必ず開始タグと終了タグで囲まれてないといけない
